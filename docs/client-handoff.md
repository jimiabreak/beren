# Client Handoff

## CMS Access

Your website content is managed through Sanity Studio at `[your-domain]/studio`.

### Logging In

1. Visit `[your-domain]/studio`
2. Sign in with your Sanity account
3. You'll see the Content dashboard

### Content Structure

| Section | What It Controls |
|---------|-----------------|
| **Homepage** | Hero, sections, and page builder blocks |
| **Pages** | Static pages (About, Privacy, etc.) |
| **Modular Pages** | Custom pages built with drag-and-drop sections |
| **Menu** | Categories and menu items |
| **Content Library** | Team members, testimonials, FAQ, gallery |
| **Promo Banners** | Site-wide promotional banners |
| **Redirects** | URL redirects (301/302) |
| **Site Settings** | Business info, logo, hours, social links, SEO |

### Common Tasks

**Editing a page:** Click the page name, edit fields, then click "Publish".

**Adding a new modular page:** Go to Modular Pages > Create new > Add a title and slug > Add sections in the Content tab.

**Updating business hours:** Site > Settings > Opening Hours.

**Adding a team member:** Content Library > Team > Create new.

**Setting up a redirect:** Redirects > Create new > Enter source path and destination.

## Deployment

Changes publish automatically via webhook revalidation. After clicking "Publish" in Sanity, changes appear on the live site within seconds.
